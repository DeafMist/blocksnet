masterplan_tools.method.balancing.balancer module#

This module contains methods for calculating a new feasibility profile for the selected area.

The maximisation parameter is the number of inhabitants.

masterplan_tools.method.balancing.balancer.balance_data(gdf, polygon, school, kindergarten, greening)[source]#

This function balances data about blocks in a city by intersecting the given GeoDataFrame with a polygon and calculating various statistics.

Args:

gdf (GeoDataFrame): A GeoDataFrame containing information about blocks in the city. polygon (gpd.GeoSeries): A polygon representing the area to intersect with the blocks. school (GeoDataFrame): A GeoDataFrame containing information about schools in the city. kindergarten (GeoDataFrame): A GeoDataFrame containing information about kindergartens in the city. greening (GeoDataFrame): A GeoDataFrame containing information about green spaces in the city.

Returns:

dict: A dictionary containing balanced data about blocks in the city.
